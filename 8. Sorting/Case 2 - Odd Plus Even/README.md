# Odd Minus Even

## Introduction

This case requires you to sort amount of integer in ascending, and sum all the numbers with odd index and even index. And then print out the sum of numbers with odd index minus sum of numbers with event index

## Input

the input will start from _t_ determining the test case. For each _t_ lines, there will be an integer _n_ determining total numbers, for each _n_ lines, the input will be _Arr[i]_

## Output

Print the sum of numbers with odd index minus sum of numbers with event index. But followed by "Case #_i_: "

## Test Case

Input

<pre>
2
5
4
2
5
2
1
1
1
</pre>

Output

<pre>
Case #1: -2
Case #2: 1
</pre>
